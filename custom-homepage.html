Datos Accesibles

El repositorio de datos de investigación proporciona una plataforma para que los investigadores afiliados a USACH compartan, conserven, citen y hagan que los datos de investigación sean accesibles y detectables. El repositorio de datos de investigación de USACH proporciona un identificador digital permanente para los datos de investigación, que cumple con las políticas de intercambio de datos. El repositorio funciona con la aplicación de código abierto Dataverse, desarrollada y utilizada por la Universidad de Harvard. Tanto el Repositorio de datos de investigación de USACH como el Repositorio institucional KEEP son administrados por la Biblioteca de USACH para garantizar que la investigación producida en la Universidad Santiago de Chile sea reconocible y accesible para la comunidad global.
